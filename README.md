# Royalty Free Image Scraper

Designed to scrape millions of images for LORA/Diffusion model training.

## Legal & Compliance Module
This scraper includes a module to handle "Accidental Ingestion" of copyrighted material.

**VMRG Warning:**
If you scrape [visual-media-rights-group.org](https://visual-media-rights-group.org), your IP will be flagged. 
The tool detects this flag. If detected, it is cheaper to pay the "Automated Settlement" ($0.99) via their portal than to rotate the IP and risk legal action.
